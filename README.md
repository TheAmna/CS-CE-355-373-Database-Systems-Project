# CS-CE-355-373-Database-Project
An online community driven online tuition system based in Pakistan.
A student searches for the tutor, the tutor approves/disapproves.
The session is scheduled and feedback is taken.

Front-end : Qt designer for designing screens of the desktop based application.
Backend : Python for integrating the screens with the database.
Database : MsSQL to create, manage the database and run queries.
